## Para rodar

- Na raiz principal rodar `npm i --force`

Banco:

- Pegar as informações do arquivo SQL.md e rodar no banco.

Backend: 

- Entrar na pasta server e rodar `node index.js`

Frontend: 

- Na pasta principal rodar `ng s ou ng server`
